├─────────────────────────────────────────────────────────┤

┌─────────────────────────────────────────────────────────┐
│  ⌨️  全局快捷键（默认，可在设置中自定义）               │
├─────────────────────────────────────────────────────────┤
│  Ctrl+Space     → 开始/暂停                             │
│  Ctrl+R         → 重置计时                              │
│  Ctrl+1         → 设置3分钟                             │
│  Ctrl+2         → 设置4分钟                             │
│  Ctrl+3         → 设置6分钟                             │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  🖱️  鼠标操作                                           │
├─────────────────────────────────────────────────────────┤
│  左键拖动红色条  → 移动窗口                             │
│  鼠标悬停        → 显示控制面板                         │
│  点击⚙️按钮      → 切换控制面板（固定）                │
│  右键点击        → 显示快捷菜单                         │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  🎨 状态颜色                                            │
├─────────────────────────────────────────────────────────┤
│  🟢 绿色         → 计时进行中                           │
│  ⚪ 灰色         → 已暂停                               │
│  🔴 红色(闪烁)   → 超时警告                             │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  ⚙️  功能特性                                           │
├─────────────────────────────────────────────────────────┤
│  ✅ 全屏PPT上方显示                                     │
│  ✅ 屏幕角落自动吸附                                    │
│  ✅ 背景透明度可调(20-100%)                             │
│  ✅ 时间显示保持清晰                                    │
│  ✅ 超时后红色正计时                                    │
│  ✅ 系统托盘最小化                                      │
│  ✅ 自定义快捷键                                        │
│  ✅ 低资源占用                                          │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  📝 常用时间设置                                        │
├─────────────────────────────────────────────────────────┤
│  演讲练习    → 3分钟  (按1)                             │
│  快速冲刺    → 4分钟  (按2)                             │
│  短任务      → 6分钟  (按3)                             │
│  番茄工作法  → 25分钟 (自定义输入)                      │
│  考试模拟    → 根据需要自定义                           │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  🎯 使用建议                                            │
├─────────────────────────────────────────────────────────┤
│  演讲场景    → 右上角, 透明度70-80%                     │
│  学习场景    → 右下角, 透明度90-100%                    │
│  运动场景    → 屏幕中央, 透明度80%                      │
│  厨房场景    → 任意角落, 透明度100%                     │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  🔧 系统托盘操作                                        │
├─────────────────────────────────────────────────────────┤
│  左键点击    → 显示/隐藏窗口                            │
│  右键点击    → 显示托盘菜单                             │
│    ├─ 显示   → 显示窗口                                 │
│    ├─ 隐藏   → 隐藏窗口                                 │
│    └─ 退出   → 完全退出应用                             │
└─────────────────────────────────────────────────────────┘

│  • 某些全屏游戏可能需要窗口化                           │
│  • 拖动时靠近屏幕角落会自动吸附                         │
