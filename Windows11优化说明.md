## 🎯 全屏应用支持

### PowerPoint全屏模式

在Windows 11上，应用使用了最高级别的窗口置顶：

```javascript
mainWindow.setAlwaysOnTop(true, 'screen-saver', 1);
mainWindow.setVisibleOnAllWorkspaces(true, { 
  visibleOnFullScreen: true 
});
```

**支持的场景：**
- ✅ PowerPoint幻灯片放映
- ✅ Adobe Acrobat全屏
- ✅ 浏览器全屏（F11）
- ✅ 视频播放器全屏
- ✅ 多数应用的全屏模式

**可能不支持的场景：**
- ❌ 某些独占全屏游戏
- ❌ 部分DirectX全屏应用
- ⚠️ 需要独占模式的专业软件

## 🔧 故障排查

### Windows 11 特定问题

#### 1. 窗口不置顶

**症状：** 全屏应用时看不到计时器

**解决方案：**
```
1. 检查Windows 11版本（建议23H2及以上）
2. 重启应用
3. 检查应用是否独占全屏（DirectX）
4. 尝试窗口化运行该应用
```

#### 2. 透明效果不生效

**症状：** 窗口不透明

**解决方案：**
```
1. 检查Windows设置：
   设置 → 个性化 → 颜色 → 透明效果（开启）
2. 检查显卡驱动更新
3. 关闭省电模式
```

#### 3. 拖动卡顿

**症状：** 拖动窗口不流畅

**解决方案：**
```
1. 关闭其他占用GPU的应用
2. 更新显卡驱动
3. 降低透明度到100%
4. 检查Windows动画设置
```

#### 4. 快捷键失效

**症状：** 全局快捷键不响应

**解决方案：**
```
1. 检查其他应用是否占用相同快捷键
2. 关闭游戏模式（Xbox Game Bar）
3. 重启应用
4. 以管理员身份运行
```

## 📈 Windows 11 优化建议

### 系统设置优化

为获得最佳体验，建议以下设置：

