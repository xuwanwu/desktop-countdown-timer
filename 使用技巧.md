# 使用技巧和最佳实践

## 🎯 演讲场景最佳实践

### 演讲前准备

1. **设置位置**
   - 拖动窗口到右上角或右下角
   - 避免遮挡重要内容
   - 建议右上角：方便查看时间

2. **调整透明度**
   - 全屏PPT建议：70-80%
   - 桌面使用建议：90-100%
   - 根据背景颜色调整

3. **预设时间**
   - 提前设置好演讲时长
   - 测试一次确保正常工作
   - 记住快捷键（1/2/3）

### 演讲中技巧

1. **无干扰启动**
   - 进入演讲模式前启动计时
   - 使用空格键暂停/开始
   - 不要频繁查看，保持眼神交流

2. **时间管理**
   - 绿色：正常进行
   - 提前1分钟注意
   - 红色：立即总结

3. **灵活调整**
   - 按R快速重置
   - 按1/2/3切换时长
   - 右键菜单快速操作

## 📚 学习场景使用

### 番茄工作法

1. **标准番茄钟**
   ```
   工作：25分钟 → 自定义输入"25"
   休息：5分钟  → 自定义输入"5"
   ```

2. **个性化番茄钟**
   - 短番茄：15分钟（适合新手）
   - 长番茄：45分钟（深度工作）
   - 使用超时功能作为缓冲时间

### 考试模拟

1. **设置考试时长**
   - 输入精确分钟数
   - 开始前深呼吸
   - 超时后停止答题

2. **时间分配**
   - 观察进度条把握节奏
   - 1/3时间检查进度
   - 最后5分钟检查

## 🏃 运动场景

### HIIT训练

1. **间歇计时**
   - 运动：3分钟（按1）
   - 休息：4分钟（按2）
   - 快速切换无需操作

2. **放置位置**
   - 屏幕中央偏上
   - 透明度80%
   - 确保运动时可见

## 🍳 日常生活

### 厨房计时

1. **煮蛋计时**
   - 溏心蛋：6分钟（按3）
   - 全熟蛋：自定义10分钟

2. **烹饪提醒**
   - 设置多个计时
   - 超时后有明显提示
   - 放在厨房可视位置

### 洗衣计时

1. **洗衣机**
   - 自定义30-60分钟
   - 超时后红色提醒
   - 不会错过洗好的衣服

## 💡 高级技巧

### 多任务管理

1. **重叠任务**
   - 任务A计时中
   - 记住剩余时间
   - 重置后开始任务B
   - 返回任务A继续

2. **时间记录**
   - 观察超时时间
   - 评估任务实际耗时
   - 优化下次时间设置

### 专注力训练

1. **逐步增加**
   ```
   第一周：3分钟专注
   第二周：4分钟专注
   第三周：6分钟专注
   ```

2. **分心记录**
   - 每次分心按暂停
   - 记录暂停次数
   - 逐步减少暂停

### 组合使用

1. **双屏场景**
   - 主屏：工作内容
   - 副屏：计时器
   - 互不干扰

2. **多显示器**
   - 拖动到不同屏幕
   - 角落自动吸附
   - 每个屏幕不同用途

## ⚙️ 性能优化

### 低配置电脑

1. **降低透明度**
   - 设置为100%不透明
   - 减少GPU负担

2. **减少拖动**
   - 启动后固定位置
   - 避免频繁移动

### 全屏应用兼容

1. **测试兼容性**
   - 提前测试全屏应用
   - 确认置顶功能正常
   - 必要时调整透明度

2. **游戏场景**
   - 窗口化游戏：完美兼容
   - 全屏游戏：可能需要切换
   - 根据游戏设置调整

## 🎨 个性化建议

### 不同场景透明度

| 场景 | 建议透明度 | 原因 |
|------|-----------|------|
| PPT全屏 | 70-80% | 不遮挡内容 |
| 视频播放 | 60-70% | 尽量透明 |
| 代码编辑 | 80-90% | 清晰可见 |
| 网页浏览 | 90-100% | 清晰易读 |

### 不同任务时长

| 任务类型 | 建议时长 | 说明 |
|---------|---------|------|
| 快速冲刺 | 3-4分钟 | 高强度专注 |
| 标准工作 | 15-25分钟 | 番茄工作法 |
| 深度工作 | 45-60分钟 | 需要进入心流 |
| 会议发言 | 3-6分钟 | 避免超时 |

## 🔧 故障排除

### 快捷键不生效

1. **检查冲突**
   - 关闭其他应用
   - 检查系统快捷键
   - 尝试单个按键

2. **重启应用**
   - 退出托盘应用
   - 重新启动
   - 测试快捷键

### 窗口被遮挡

1. **检查置顶**
   - 应用应该自动置顶
   - 某些特殊应用可能例外
   - 尝试调整位置

2. **特殊应用**
   - 全屏视频播放器
   - 某些游戏
   - 专业演示软件

### 超时不明显

1. **增强提示**
   - 降低透明度到100%
   - 确保窗口在视野内
   - 红色闪烁很明显

2. **配合其他提醒**
   - 手机备用提醒
   - 系统音效（未来版本）
   - 振动提醒（手机同步）
