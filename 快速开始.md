# 🚀 快速开始 - 3步打包成安装文件

想要把应用打包成 `.exe` 安装文件？跟着做就对了！

## 📦 超简单3步走

### 第1步：安装依赖（首次需要）⏱️ 5分钟

双击运行：`install.bat`

或在命令行输入：
```bash
npm install
```

等待安装完成，看到"安装完成"就OK了！

---

### 第2步：准备图标（可选）⏱️ 2分钟

**方法A：用工具生成（推荐）**
1. 双击 `创建简单图标.html`
2. 选一个好看的图标
3. 点"下载图标"
4. 把下载的文件放到 `assets` 文件夹，确保命名为 `icon.png`

**方法B：跳过这步**
- 没图标也能打包
- 只是会用默认图标

---

### 第3步：一键打包 ⏱️ 5分钟

双击运行：`package.bat`

然后：
1. 看到菜单，输入 `1`（安装版）
2. 等待打包完成（喝杯咖啡☕）
3. 自动打开 `dist` 文件夹
4. 看到 `倒计时小组件-1.0.0-Setup.exe`

**🎉 完成！这就是你的安装文件！**

---

## 💡 快速问答

### Q: 打包的文件有多大？
A: 约60MB，包含了完整的运行环境

### Q: 用户需要安装什么吗？
A: 不需要！双击安装就能用

### Q: 打包要多久？
A: 首次约10分钟，之后约3-5分钟

### Q: 出现错误怎么办？
A: 查看 `一键打包说明.md` 的常见问题部分

### Q: 可以在其他电脑运行吗？
A: 可以！Windows 10/11都能运行

## 📝 完整流程一览

```
1. 双击 install.bat        ← 安装依赖
   ↓

   ↓

   ↓
5. 输入 1                  ← 选择安装版
   ↓
6. 等待完成                ← 约5分钟
   ↓
7. 打开 dist 文件夹        ← 看到安装文件
   ↓
8. 分享给朋友！            ← 大功告成！
```

## 🎯 给用户安装

把 `dist/倒计时小组件-1.0.0-Setup.exe` 发给用户

用户操作：
1. 双击 Setup.exe
2. (如果出现警告) 点"更多信息" → "仍要运行"  
3. 下一步 → 安装
4. 完成！

## 🔥 常见错误速查

| 错误 | 解决方案 |
|------|---------|
| Cannot find module | 运行 `install.bat` |
| icon not found | 跳过图标或添加 `assets/icon.png` |
| 打包很慢 | 正常，首次需下载文件 |
| 文件太大 | 正常，包含完整环境 |
