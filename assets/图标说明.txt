图标文件说明
================

本文件夹需要一个 icon.png 文件作为应用图标。

方式一：使用我提供的简单方案（推荐）
------------------------------------
1. 将任意 PNG 图片重命名为 icon.png
2. 放在 assets 文件夹中
3. 建议尺寸：256x256 像素或 512x512 像素

如果没有图标，可以使用纯色占位符：
- 打开画图软件（Windows自带）
- 创建 256x256 像素的新图片
- 填充红色背景
- 写上白色文字 "T" 或 "倒计时"
- 保存为 icon.png

方式二：在线生成图标
--------------------
访问以下网站免费生成图标：
- https://icon.kitchen/  （推荐，简单易用）
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

生成后下载 PNG 格式，重命名为 icon.png

方式三：专业设计
----------------
如果需要专业图标：
1. 使用 Figma / Sketch / Adobe Illustrator 设计
2. 导出为 PNG 格式
3. 尺寸：512x512 像素
4. 透明背景
5. 保存为 icon.png

图标设计建议
------------
- 简洁明了，一眼能看出是计时器
- 使用红色/白色配色，与应用主题一致
- 可以包含时钟、沙漏、倒计时等元素
- 背景建议透明或圆角矩形

临时解决方案
------------
如果暂时没有图标，打包时会使用 Electron 默认图标。
应用功能不受影响，只是图标不够美观。

重要提示
--------
图标文件必须命名为：icon.png
位置：assets/icon.png
